# sports_image_classification

cnn모델을 이용하여 sports image 데이터를 수집하여 학습시키고, 학습한 모델로 sports image를 분류한다.

1단계) 데이터 수집
class : 농구, 축구, 배구, 야구, 골프, 탁구
train set : 각 300장
test set : 각 100장

2단계) 학습 모델 구성

3단계) validaion 코드 추가
validation set : train set의 20%

4단계) loss 비교

5단계) data augmentation

6단계) hyper parameter
